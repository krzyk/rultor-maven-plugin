# rultor-maven-plugin

Add it to your list of plugins:

```xml
<project>
  <build>
    <plugins>
      <plugin>
        <groupId>com.rultor</groupId>
        <artifactId>rultor-maven-plugin</artifactId>
        <version>0.1</version>
      </plugin>
    </plugins>
  </build>
</project>
```

More details: [maven.rultor.com](http://maven.rultor.com)
